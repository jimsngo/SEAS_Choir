🎵 SEAS Choir Music Resource Repository

Welcome to the SEAS Choir music management system. This repository manages choir music resources including lyrics, metadata, PDFs, and automation scripts.

📚 Documentation

For detailed information about the project structure, automation workflows, and how to use the system, see README.md

🖥️ Server Application

The main application runs on a local Node.js server. To start:

  • Run: ./shell-scripts/run_script.sh
  • Public Page: http://localhost:3000
  • Admin Panel: http://localhost:3000/server
